Overview

The base class for all SMART server documents.

This class implements the basics for URL requests, which are all funnelled via performMethod:withBody:orParameters:ofType:httpMethod:callback:.

performMethod:withBody:orParameters:ofType:httpMethod:callback:

The basic method to perform REST methods on the server with App credentials.

- (void)performMethod:(NSString *)aMethod withBody:(id)body orParameters:(NSArray *)parameters ofType:(NSString *)contentType httpMethod:(NSString *)httpMethod callback:(SMSuccessRetvalueBlock)callback

Parameters

aMethod

The path to call on the server

body

The body data (NSData) or string (NSString)

parameters

An array full of strings in the form “key=value”

contentType

The optional content-type for PUT and POST

httpMethod

The http method, for now GET, PUT or POST

callback

A block to execute when the call has finished

Discussion

All convenience methods invoke this method. It uses a SMServerCall instance to handle the loading; SMServerCall only allows a body string or parameters, but not both, with the body string taking precedence if both are present.
