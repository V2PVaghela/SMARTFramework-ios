Overview

A class representing “http://smartplatforms.org/terms#ScratchpadData” objects.

The scratchpad lets you store any data on a per-app-per-record basis. You have to decide on the format yourself and also do the parsing. To download the data use “get:”, to upload the data set the ivars and use “store:” and to delete scratchpad data use “delete:”.

Example usage:

[record.scratchpad get:^(BOOL userDidCancel, NSString *__autoreleasing errorMessage) {

    NSLog(@"Scratchpad: %@  (%@)", record.scratchpad.stringValue, record.scratchpad.contentType);

}];
